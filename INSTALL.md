# Installation Instructions

Run this command to install all dependencies:

```powershell
npm install
```

This will install:
- React & React DOM
- TypeScript & types
- Zustand (state management)
- Ethers.js (blockchain)
- DID libraries (did-jwt, ethr-did, etc.)
- Webpack & build tools
- All other required packages

After installation completes, follow SETUP.md for next steps.
